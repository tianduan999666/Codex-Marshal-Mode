```text
Codex-Chancellor-Mode
├─ README.md                     # 新仓入口
├─ AGENTS.md                     # 工作规则
├─ .gitignore                    # 忽略策略
├─ docs/                         # 正式文档
├─ .codex/                       # 运行态任务包
├─ logs/                         # 运行留痕
├─ temp/                         # 生成物快照与历史归档
```

```text

```

## 目录角色

- `docs/`：说清楚要做什么、为什么这样做
- `.codex/`：承载试运行任务包，不把任务状态丢在聊天里
- `logs/`：留痕，不放正式规则
- `temp/generated/`：生成物快照
- `temp/legacy/`：历史归档
- `codex-home-export/`：本机生产母体最小闭环，不承接旧仓整包内容

## 当前不建的目录

- 不急着建 `tests/`
- 不急着建 `tools/`
- 不急着建 `dist/`

其中 `codex-home-export/` 已因“本机生产切换最小闭环”获本轮批准；其余目录仍等新 V4 的最小方案拍板后，再按需增加。
