# V4 面板传令 1 页速查卡

最后更新：2026-04-05
状态：现行速查件
适用范围：官方 Codex 面板里的日常高频输入
说明：本卡只保留最常用的 4 句，不展开维护层细节。

## 一句话结论

日常只记 4 句：`传令：继续`、`传令：状态`、`传令：版本`、`传令：修一下登录页`。

## 4 句高频传令

| 你输入什么 | 什么时候用 | 你期待看到什么 |
| --- | --- | --- |
| `传令：继续` | 继续上一轮正在做的事 | 若有激活任务，优先续上，不重复新建 |
| `传令：状态` | 先看当前是否稳定 | 返回固定 6 行：`版本 / 上次检查 / 自动修复 / 关键文件一致性 / 当前模式 / 当前任务` |
| `传令：版本` | 先看现在跑的是哪版 | 返回固定 3 行：`版本号 / 版本来源 / 真源路径` |
| `传令：修一下登录页` | 直接开工一个自然语言任务 | 丞相按单入口主链开始执行；必要时先轻检、修平、建任务 |

## 最短记忆法

```text
要继续：传令：继续
要看状态：传令：状态
要看版本：传令：版本
要干活：传令：修一下登录页
```

## 推荐使用顺序

```text
不确定稳不稳
  → 传令：状态

不确定是不是最新版
  → 传令：版本

知道要接着干
  → 传令：继续

知道要开始新任务
  → 传令：修一下登录页
```

## 常见误区

| 误区 | 正确做法 |
| --- | --- |
| 一上来先讲一大段维护流程 | 先用 `传令：XXXX`，必要时再回维护层 |
| 每次都想手动自检 | 默认不用；只有首次、异常、维护时才补 |
| 把命令体系越记越多 | 日常就记这 4 句 |
| 想先把丞相模式全做完再开工 | 不等，直接用当前稳定主链推进业务 |

## 维护层只在必要时使用

如果你真的需要维护动作，再回终端记这 4 个：

```powershell
.\install.cmd
%USERPROFILE%\.codex\upgrade.cmd
%USERPROFILE%\.codex\self-check.cmd
%USERPROFILE%\.codex\rollback.cmd
```

如需一次看全漂移，只在维护时执行：

```powershell
%USERPROFILE%\.codex\self-check.cmd -MaintainerMode
```
