## 三阶段改造（按主公新口径）

### Phase 1：100 个重度 Codex 用户能上手

只做降门槛，不跳去做 UI。

1. 一键初始化：10 分钟内跑通第一个任务。
2. 一键新任务：把任务包 5 件套收敛为一条命令。
3. 10 分钟上手指南：全程不用术语，默认以 Codex 面板为入口。
4. 公开试用文案：先拿到第一批真实回复，验证是否有人愿意试用。

### Phase 2：1000 个开发者能用

1. 做跨平台 CLI，但仍保持 Codex 面板为默认日常入口。
2. 把门禁变成无感知的人话提示。
3. 把会话恢复做成一键可粘贴模板。
4. 再谈免费版 / 付费版边界。

### Phase 3：10000 个 AI 用户能用

1. 再做极简 Web 界面。
2. 再去掉对 git 的强依赖。
3. 再考虑多入口与更大众化场景。
