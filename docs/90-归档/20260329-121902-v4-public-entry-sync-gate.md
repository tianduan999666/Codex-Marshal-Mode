## 结果

- 当前自动门禁不只校验运行态混入与跟踪面，还会校验执行区现行标准件在四处公开入口中的同步一致性。
- 当前仓现状可通过新门禁；人为删掉一条执行区入口时，门禁会自动阻断。

## 理由

- 公开口径漂移最常见的不是“文件丢了”，而是“现行标准件新增或调整后，入口没同步”。
- 先把执行区现行标准件入口四处同步收成自动门禁，收益高、误伤低、最符合最小能量推进。
- 这一步仍保持当前目录内自含，不新增目录、不引入外部依赖。
