# 丞相模式（Chancellor Mode）

```powershell
git clone https://github.com/tianduan999666/Codex-Chancellor-Mode.git
cd Codex-Chancellor-Mode
.\install.cmd
```

普通用户只做这 3 行。默认不会覆盖你现有的 `~/.codex/config.toml`。
装完后回官方 Codex 面板，依次输入：

- `传令：版本`
- `传令：状态`
- `传令：修一下登录页`

如果准备切到新聊天继续，先在当前会话输入：

- `传令：交班`

新聊天只需输入：

- `传令：接班`

维护说明见：`docs/README.md`
